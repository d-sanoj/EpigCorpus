# Data

This folder contains the EDCS inscription data exports generated by the scraper.

## Files

- `edcs_inscriptions.json` - JSON format export
- `edcs_inscriptions.tsv` - Tab-separated values format export
- `edcs_checkpoint.json` - Temporary checkpoint file (not tracked in git)

## Generation

To generate these data files, run the scraper from the project root:

```bash
python src/edcs_scraper.py
```

This will create or update the JSON and TSV exports with new inscriptions from the EDCS database.

## Notes

- Data files are generated on-demand by running the scraper
- Checkpoint file is used for resuming interrupted scraping runs
- Each run incrementally appends only new records from the website
